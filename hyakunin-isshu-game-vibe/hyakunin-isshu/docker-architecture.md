# Dockerコンテナ構成図

```
┌─────────────────────────────────────────────────────────┐
│                      ホストマシン (macOS)                    │
│                                                         │
│  ┌───────────────────────────────────────────────────┐  │
│  │           Docker Compose                          │  │
│  │                                                   │  │
│  │  ┌─────────────────────────────────────────────┐ │  │
│  │  │  hyakunin-isshu-web-1 コンテナ              │ │  │
│  │  │                                             │ │  │
│  │  │  ベースイメージ: python:3.12-slim            │ │  │
│  │  │                                             │ │  │
│  │  │  ┌───────────────────────────────────────┐ │ │  │
│  │  │  │  Flask アプリケーション                │ │ │  │
│  │  │  │  - app.py                            │ │ │  │
│  │  │  │  - ポート: 5000 (コンテナ内)          │ │ │  │
│  │  │  └───────────────────────────────────────┘ │ │  │
│  │  │                                             │ │  │
│  │  └─────────────────────────────────────────────┘ │  │
│  │                      ↕                            │  │
│  │              ポートマッピング                        │  │
│  │            5001 (ホスト) → 5000 (コンテナ)          │  │
│  └───────────────────────────────────────────────────┘  │
│                      ↕                                  │
│              http://localhost:5001                      │
└─────────────────────────────────────────────────────────┘
```

## 構成の詳細

- **サービス名**: web
- **イメージ**: hyakunin-isshu-web (カスタムビルド)
- **ベースイメージ**: Python 3.12 Slim
- **依存関係**: Flask 3.1.0
- **ポートマッピング**: ホストの5001番ポート → コンテナの5000番ポート
- **エントリーポイント**: `python app.py`
- **作業ディレクトリ**: `/app`
